\doxysubsection{Detailed Description}
This file contains the implementation of solvers for Euler equations and Navier-\/\+Stokes equations. 

The solvers include methods for evaluating time steps, applying boundary conditions, and solving the equations using explicit, implicit, or Runge-\/\+Kutta methods. The file also includes logic for handling viscous and inviscid flows, as well as error estimation and solution updates.\hypertarget{_solver_8cpp_autotoc_md0}{}\doxysubsubsection{\texorpdfstring{Key Functions\+:}{Key Functions\+:}}\label{_solver_8cpp_autotoc_md0}

\begin{DoxyItemize}
\item {\bfseries{Evaluate\+\_\+\+Time\+\_\+\+Step}}\+: Determines the time step for a given cell based on whether the flow is viscous or inviscid.
\item {\bfseries{Inviscid\+\_\+\+Solver}}\+: Core logic for solving Euler equations for inviscid flows.
\item {\bfseries{Viscous\+\_\+\+Solver}}\+: Core logic for solving Navier-\/\+Stokes equations for viscous flows.
\item {\bfseries{Viscous\+\_\+\+Time\+\_\+\+Step\+\_\+X}}\+: Functions for evaluating time steps under different viscous flow conditions.
\item {\bfseries{Inviscid\+\_\+\+Time\+\_\+\+Step}}\+: Function for evaluating time steps for inviscid flows.
\end{DoxyItemize}\hypertarget{_solver_8cpp_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{Euler Equations\+:}{Euler Equations\+:}}\label{_solver_8cpp_autotoc_md1}
The Euler equations describe the motion of an inviscid fluid and are solved using the {\ttfamily Inviscid\+\_\+\+Solver} function. The solver applies boundary conditions, evaluates convective fluxes, and updates the solution iteratively.\hypertarget{_solver_8cpp_autotoc_md2}{}\doxysubsubsubsection{\texorpdfstring{Key Steps in the Euler Solver\+:}{Key Steps in the Euler Solver\+:}}\label{_solver_8cpp_autotoc_md2}

\begin{DoxyEnumerate}
\item {\bfseries{Boundary Conditions}}\+: Apply boundary conditions to ensure physical consistency.
\item {\bfseries{Flux Evaluation}}\+: Compute convective fluxes for all cells.
\item {\bfseries{Time Integration}}\+: Use explicit or implicit methods to integrate the equations in time.
\item {\bfseries{Error Estimation}}\+: Estimate the error in the solution to monitor convergence.
\item {\bfseries{Solution Update}}\+: Update the conservative variables based on the computed fluxes.
\end{DoxyEnumerate}\hypertarget{_solver_8cpp_autotoc_md3}{}\doxysubsubsubsection{\texorpdfstring{Time Step Calculation\+:}{Time Step Calculation\+:}}\label{_solver_8cpp_autotoc_md3}
The time step is calculated based on the CFL condition, ensuring numerical stability. The maximum eigenvalues in the x and y directions are used to compute the time step for each cell.\hypertarget{_solver_8cpp_autotoc_md4}{}\doxysubsubsection{\texorpdfstring{Viscous Flows\+:}{Viscous Flows\+:}}\label{_solver_8cpp_autotoc_md4}
For viscous flows, the Navier-\/\+Stokes equations are solved using the {\ttfamily Viscous\+\_\+\+Solver} function. Additional viscous fluxes are computed, and the solver supports second-\/order accuracy.\hypertarget{_solver_8cpp_autotoc_md5}{}\doxysubsubsubsection{\texorpdfstring{Key Features\+:}{Key Features\+:}}\label{_solver_8cpp_autotoc_md5}

\begin{DoxyItemize}
\item Support for multiple viscous time step evaluation methods ({\ttfamily Viscous\+\_\+\+Time\+\_\+\+Step\+\_\+1}, {\ttfamily Viscous\+\_\+\+Time\+\_\+\+Step\+\_\+2}, {\ttfamily Viscous\+\_\+\+Time\+\_\+\+Step\+\_\+3}).
\item Evaluation of wall skin friction coefficients for viscous walls.
\item Handling of both time-\/accurate and steady-\/state simulations.
\end{DoxyItemize}\hypertarget{_solver_8cpp_autotoc_md6}{}\doxysubsubsection{\texorpdfstring{Output\+:}{Output\+:}}\label{_solver_8cpp_autotoc_md6}
The solver outputs the following\+:
\begin{DoxyItemize}
\item Iteration details, including time step, errors, and wall clock time.
\item Solution files containing the computed flow variables.
\item Skin friction coefficients for viscous walls.
\end{DoxyItemize}\hypertarget{_solver_8cpp_autotoc_md7}{}\doxysubsubsection{\texorpdfstring{Notes\+:}{Notes\+:}}\label{_solver_8cpp_autotoc_md7}

\begin{DoxyItemize}
\item The solver is designed to handle both 2D and 3D grids.
\item The implementation assumes structured grid data with precomputed face normals and areas.
\item The CFL number and other parameters must be set appropriately for the specific test case. 
\end{DoxyItemize}
