Computes the Min\+Mod limiter for two input values. 

The Min\+Mod limiter is a slope limiter function used in numerical methods to ensure stability and prevent oscillations in solutions. It takes two input values, compares their magnitudes, and determines the limited value based on their signs and the smaller magnitude.

Computes the Min\+Mod limiter for three input values and updates the result in the provided reference. 

The Min\+Mod function is used in numerical methods, particularly in finite volume and finite difference schemes, to compute a slope limiter. It ensures that the solution remains monotonic by selecting the minimum magnitude value among the inputs if they all have the same sign. If the inputs have mixed signs, the limiter is set to zero.


\begin{DoxyParams}{Parameters}
{\em a} & Reference to the first input value. \\
\hline
{\em b} & Reference to the second input value. \\
\hline
{\em c} & Reference to the third input value. \\
\hline
{\em phi} & Reference to the output value, which will be updated based on the Min\+Mod calculation.\\
\hline
\end{DoxyParams}
The function uses the following logic\+:
\begin{DoxyItemize}
\item If all inputs have the same sign and are non-\/zero, the minimum magnitude value is selected.
\item If all inputs are negative, the maximum magnitude value is selected.
\item If the inputs have mixed signs, the output is set to zero. 
\end{DoxyItemize}

void Second\+\_\+\+Order\+\_\+\+Limiter (\begin{DoxyParamCaption}\item[{const int \&}]{Cell\+\_\+\+Index}{, }\item[{const int \&}]{Face\+\_\+\+No}{, }\item[{vector$<$ double $>$ \&}]{d\+\_\+U}{}\end{DoxyParamCaption})}



Computes the second-\/order limited slopes and reconstructs variables for a given cell and face. 

This function calculates the limited slopes using the Min\+Mod limiter and reconstructs the left and right states of the variables at the cell interface. It is used in finite volume methods to ensure stability and accuracy of the solution.


\begin{DoxyParams}{Parameters}
{\em Cell\+\_\+\+Index} & The index of the current cell. \\
\hline
{\em Face\+\_\+\+No} & The face number (0, 1, 2, or 3) corresponding to the interface direction. \\
\hline
{\em d\+\_\+U} & A vector to store the difference between reconstructed right and left states of variables.\\
\hline
\end{DoxyParams}
The function performs the following steps\+:
\begin{DoxyEnumerate}
\item Identifies neighboring cells and computes distances between cell centers.
\item Computes slopes using the specified slope method (Min\+Mod with two or three arguments).
\item Applies the limiter to the computed slopes.
\item Reconstructs the left and right states of variables at the interface.
\end{DoxyEnumerate}

The function uses the following helper lambdas\+:
\begin{DoxyItemize}
\item compute\+Slopes\+: Computes the slopes for the current variable.
\item apply\+Limiter\+: Applies the Min\+Mod limiter to the computed slopes.
\item reconstruct\+Variables\+: Reconstructs the left and right states of the variable at the interface.
\end{DoxyItemize}

The function supports four face directions\+:
\begin{DoxyItemize}
\item Face\+\_\+\+No = 0\+: i-\/1/2, j interface
\item Face\+\_\+\+No = 1\+: i, j-\/1/2 interface
\item Face\+\_\+\+No = 2\+: i+1/2, j interface
\item Face\+\_\+\+No = 3\+: i, j+1/2 interface
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The function assumes that global variables such as {\ttfamily U\+\_\+\+Cells}, {\ttfamily Cells}, {\ttfamily Limiter\+\_\+\+Zeta}, and {\ttfamily Limiter\+\_\+\+Zeta1} are defined and accessible. It also assumes that the {\ttfamily Min\+Mod} function is implemented elsewhere. 
\end{DoxyNote}
