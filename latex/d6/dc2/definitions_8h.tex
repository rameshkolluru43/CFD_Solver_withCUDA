\doxysubsection{Detailed Description}
Header file containing global constants, structures, and function declarations for the solver. 

This file includes all necessary libraries, defines global constants, declares global variables, and provides function prototypes required for the solver. It also contains detailed documentation for the structures and functions used in the computational domain and boundary condition handling.\hypertarget{definitions_8h_Libraries}{}\doxysubsection{\texorpdfstring{Libraries}{Libraries}}\label{definitions_8h_Libraries}
Includes standard C++ libraries, VTK libraries, Boost, and JSONCPP for various functionalities.\hypertarget{definitions_8h_Global}{}\doxysubsection{\texorpdfstring{Constants}{Constants}}\label{definitions_8h_Global}
Defines mathematical constants, physical constants, and other macros used throughout the code.\hypertarget{definitions_8h_Structures}{}\doxysubsection{\texorpdfstring{Structures}{Structures}}\label{definitions_8h_Structures}
