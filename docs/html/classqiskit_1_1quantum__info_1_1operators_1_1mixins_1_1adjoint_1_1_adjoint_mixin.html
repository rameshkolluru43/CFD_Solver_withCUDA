Detailed Description

Abstract Mixin for operator adjoint and transpose operations.

This class defines the following methods

    - :meth:`transpose`
    - :meth:`conjugate`
    - :meth:`adjoint`

The following abstract methods must be implemented by subclasses
using this mixin

    - ``conjugate(self)``
    - ``transpose(self)``
