bool qiskit_algorithms.eigensolvers.eigensolver.Eigensolver.supports_aux_operators ( cls) 
Whether computing the expectation value of auxiliary operators is supported.

If the eigensolver computes the eigenvalues of the main operator, then it can compute
the expectation value of the ``aux_operators`` for that state. Otherwise they will be ignored.

Returns:
    ``True`` if ``aux_operator`` expectations can be evaluated, ``False`` otherwise.

Reimplemented in qiskit_algorithms.eigensolvers.numpy_eigensolver.NumPyEigensolver, and qiskit_algorithms.eigensolvers.vqd.VQD.
