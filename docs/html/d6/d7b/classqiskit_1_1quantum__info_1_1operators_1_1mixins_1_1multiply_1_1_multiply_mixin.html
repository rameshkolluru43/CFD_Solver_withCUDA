Detailed Description

Abstract Mixin for scalar multiplication.

This class defines the following operator overloads:

    - ``*`` / ``__rmul__`
    - ``/`` / ``__truediv__``
    - ``__neg__``

The following abstract methods must be implemented by subclasses
using this mixin

    - ``_multiply(self, other)``
