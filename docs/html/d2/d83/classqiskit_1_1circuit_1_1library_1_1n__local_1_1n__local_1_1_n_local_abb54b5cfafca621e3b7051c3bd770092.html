list[Parameter] qiskit.circuit.library.n_local.n_local.NLocal.ordered_parameters ( self) 
The parameters used in the underlying circuit.

This includes float values and duplicates.

Examples:

    >>> # prepare circuit ...
    >>> print(nlocal)
