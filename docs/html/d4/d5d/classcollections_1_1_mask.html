Detailed Description

The inverse of a :class:`Projection`, masking out keys.

>>> sample = {'a': 1, 'b': 2, 'c': 3}
>>> msk = Mask(['a', 'c', 'd'], sample)
>>> dict(msk)
{'b': 2}
