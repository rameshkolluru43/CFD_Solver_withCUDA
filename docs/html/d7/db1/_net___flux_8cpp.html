Detailed Description

Contains functions for evaluating the net flux for cells in a computational domain.

This file includes implementations for calculating both first-order and second-order fluxes for all faces of a cell. It supports multiple dissipation types and handles boundary conditions. The flux calculations involve both average convective flux and dissipative flux contributions.

Functions:

  • Calculate_Flux_For_All_Faces: Computes fluxes for all faces of a given cell.
  • Evaluate_Cell_Net_Flux_1O: Evaluates first-order net flux for all cells.
  • Evaluate_Cell_Net_Flux_2O: Evaluates second-order net flux for all cells.

Dependencies:

  • Requires definitions for constants, types, and utility functions from "definitions.h".

Debugging:

  • DEBUG macro can be enabled to log entry points for flux calculations.

Notes:

  • The dissipation function is passed as a function pointer to allow flexibility in flux calculations.
  • The code supports multiple dissipation types such as LLF, MOVERS, ROE, RICCA, and MOVERS_NWSC.
