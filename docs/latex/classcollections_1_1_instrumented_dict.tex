\doxysubsection{Detailed Description}
\begin{DoxyVerb}Instrument an existing dictionary with additional
functionality, but always reference and mutate
the original dictionary.

>>> orig = {'a': 1, 'b': 2}
>>> inst = InstrumentedDict(orig)
>>> inst['a']
1
>>> inst['c'] = 3
>>> orig['c']
3
>>> inst.keys() == orig.keys()
True
\end{DoxyVerb}
