\doxysubsection{Detailed Description}
\begin{DoxyVerb}Make avaialable a limited structural pattern matching-like syntax for Python < 3.10

Patterns can be only tuples (without types) currently.
Inspired by the package pattern-matching-PEP634.

Usage:
>>> with match(args) as m:
>>>     if   m.case(('x', 'y')):
>>>         # use m.x and m.y
>>>     elif m.case(('x', 'y', 'z')):
>>>         # use m.x, m.y and m.z

Equivalent native code for Python >= 3.10:
>>> match args:
>>>     case (x, y):
>>>         # use x and y
>>>     case (x, y, z):
>>>         # use x, y and z
\end{DoxyVerb}
