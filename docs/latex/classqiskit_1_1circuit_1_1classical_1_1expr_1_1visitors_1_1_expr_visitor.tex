\doxysubsection{Detailed Description}
\begin{DoxyVerb}Base class for visitors to the :class:`Expr` tree.  Subclasses should override whichever of
the ``visit_*`` methods that they are able to handle, and should be organized such that
non-existent methods will never be called.\end{DoxyVerb}
