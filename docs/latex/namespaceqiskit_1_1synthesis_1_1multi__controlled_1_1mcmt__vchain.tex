\begin{DoxyVerb}Synthesize MCMT using a V-chain.

This uses a chain of CCX gates, using ``num_ctrl_qubits - 1`` auxiliary qubits.

For example, a 3-control and 2-target H gate will be synthesized as::

    q_0: ──■────────────────────────■──
           │                        │
    q_1: ──■────────────────────────■──
           │                        │
    q_2: ──┼────■──────────────■────┼──
           │    │  ┌───┐       │    │
    q_3: ──┼────┼──┤ H ├───────┼────┼──
           │    │  └─┬─┘┌───┐  │    │
    q_4: ──┼────┼────┼──┤ H ├──┼────┼──
         ┌─┴─┐  │    │  └─┬─┘  │  ┌─┴─┐
    q_5: ┤ X ├──■────┼────┼────■──┤ X ├
         └───┘┌─┴─┐  │    │  ┌─┴─┐└───┘
    q_6: ─────┤ X ├──■────■──┤ X ├─────
              └───┘          └───┘\end{DoxyVerb}
