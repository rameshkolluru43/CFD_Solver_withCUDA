\doxysubsection{Detailed Description}
\begin{DoxyVerb}Root base class of all nodes in the type tree.  The base case should never be instantiated
directly.

This must not be subclassed by users; subclasses form the internal data of the representation of
expressions, and it does not make sense to add more outside of Qiskit library code.\end{DoxyVerb}
