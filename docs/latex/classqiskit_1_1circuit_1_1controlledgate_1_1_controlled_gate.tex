qiskit.\+circuit.\+controlledgate.\+Controlled\+Gate.\+\_\+\+\_\+init\+\_\+\+\_\+ (\begin{DoxyParamCaption}\item[{}]{self}{, }\item[{str}]{name}{, }\item[{int}]{num\+\_\+qubits}{, }\item[{list}]{params}{, }\item[{Optional\mbox{[}str\mbox{]} }]{label}{ = {\ttfamily None}, }\item[{Optional\mbox{[}int\mbox{]} }]{num\+\_\+ctrl\+\_\+qubits}{ = {\ttfamily 1}, }\item[{Optional\mbox{[}"{}Quantum\+Circuit"{}\mbox{]} }]{definition}{ = {\ttfamily None}, }\item[{Optional\mbox{[}Union\mbox{[}int, str\mbox{]}\mbox{]} }]{ctrl\+\_\+state}{ = {\ttfamily None}, }\item[{Optional\mbox{[}\mbox{\hyperlink{classqiskit_1_1circuit_1_1gate_1_1_gate}{Gate}}\mbox{]} }]{base\+\_\+gate}{ = {\ttfamily None}, }\item[{}]{duration}{ = {\ttfamily None}, }\item[{}]{unit}{ = {\ttfamily None}, }\item[{\texorpdfstring{$\ast$}{*}}]{}{, }\item[{}]{\+\_\+base\+\_\+label}{ = {\ttfamily None}}\end{DoxyParamCaption})}

\begin{DoxyVerb}Create a new ControlledGate. In the new gate the first ``num_ctrl_qubits``
of the gate are the controls.

Args:
    name: The name of the gate.
    num_qubits: The number of qubits the gate acts on.
    params: A list of parameters for the gate.
    label: An optional label for the gate.
    num_ctrl_qubits: Number of control qubits.
    definition: A list of gate rules for implementing this gate. The
        elements of the list are tuples of (:meth:`~qiskit.circuit.Gate`, [qubit_list],
        [clbit_list]).
    ctrl_state: The control state in decimal or as
        a bitstring (e.g. '111'). If specified as a bitstring the length
        must equal num_ctrl_qubits, MSB on left. If None, use
        2**num_ctrl_qubits-1.
    base_gate: Gate object to be controlled.

Raises:
    CircuitError: If ``num_ctrl_qubits`` >= ``num_qubits``.
    CircuitError: ctrl_state < 0 or ctrl_state > 2**num_ctrl_qubits.

Examples:

Create a controlled standard gate and apply it to a circuit.
