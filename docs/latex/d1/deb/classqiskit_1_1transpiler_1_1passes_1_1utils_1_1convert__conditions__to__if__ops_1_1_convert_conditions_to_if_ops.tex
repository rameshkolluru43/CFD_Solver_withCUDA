\doxysubsection{Detailed Description}
\begin{DoxyVerb}Convert instructions whose ``condition`` attribute is set to a non-``None`` value into the
equivalent single-statement :class:`.IfElseBlock`.

This is a simple pass aimed at easing the conversion from the old style of using
:meth:`.InstructionSet.c_if` into the new style of using more complex conditional logic.\end{DoxyVerb}
