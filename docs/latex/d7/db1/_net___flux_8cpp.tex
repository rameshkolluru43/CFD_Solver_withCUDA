\doxysubsection{Detailed Description}
Contains functions for evaluating the net flux for cells in a computational domain. 

This file includes implementations for calculating both first-\/order and second-\/order fluxes for all faces of a cell. It supports multiple dissipation types and handles boundary conditions. The flux calculations involve both average convective flux and dissipative flux contributions.

Functions\+:
\begin{DoxyItemize}
\item Calculate\+\_\+\+Flux\+\_\+\+For\+\_\+\+All\+\_\+\+Faces\+: Computes fluxes for all faces of a given cell.
\item Evaluate\+\_\+\+Cell\+\_\+\+Net\+\_\+\+Flux\+\_\+1O\+: Evaluates first-\/order net flux for all cells.
\item Evaluate\+\_\+\+Cell\+\_\+\+Net\+\_\+\+Flux\+\_\+2O\+: Evaluates second-\/order net flux for all cells.
\end{DoxyItemize}

void Evaluate\+\_\+\+Cell\+\_\+\+Net\+\_\+\+Flux\+\_\+2O (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Evaluates the net flux for each cell using a second-\/order flux calculation method. 

This function iterates over all physical cells and calculates the net flux for each cell based on the specified dissipation type. It initializes the net flux for each cell to zero, applies the appropriate flux calculation method for all faces of the cell, and evaluates the time step for the current cell.

\begin{DoxyNote}{Note}
The function supports multiple dissipation types, each corresponding to a specific flux calculation method.
\end{DoxyNote}

\begin{DoxyItemize}
\item The dissipation type determines which flux calculation method is used\+:
\begin{DoxyItemize}
\item 1\+: LLF\+\_\+2O (Local Lax-\/\+Friedrichs, second-\/order)
\item 2\+: MOVERS\+\_\+2O (MOVERS, second-\/order)
\item 3\+: ROE\+\_\+2O (Roe\textquotesingle{}s method, second-\/order)
\item 4\+: RICCA\+\_\+2O (Ricca\textquotesingle{}s method, second-\/order)
\item 5\+: MOVERS\+\_\+\+NWSC\+\_\+2O (MOVERS with no wall shear correction, second-\/order)
\end{DoxyItemize}
\item The function also evaluates the time step for each cell after computing the flux.
\end{DoxyItemize}
