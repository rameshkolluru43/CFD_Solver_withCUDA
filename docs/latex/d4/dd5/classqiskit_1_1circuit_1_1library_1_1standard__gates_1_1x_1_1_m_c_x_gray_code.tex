\doxysubsection{Detailed Description}
\begin{DoxyVerb}Implement the multi-controlled X gate using the Gray code.

This delegates the implementation to the MCU1 gate, since :math:`X = H \cdot U1(\pi) \cdot H`.
\end{DoxyVerb}
