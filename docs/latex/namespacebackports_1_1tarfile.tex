{\footnotesize\ttfamily \label{namespacebackports_1_1tarfile_adf99b9f8d69d2a9723fe3c0bdd888bea} 
backports.\+tarfile.\+calc\+\_\+chksums (\begin{DoxyParamCaption}\item[{}]{buf}{}\end{DoxyParamCaption})}

\begin{DoxyVerb}Calculate the checksum for a member's header by summing up all
   characters except for the chksum field which is treated as if
   it was filled with spaces. According to the GNU tar sources,
   some tars (Sun and NeXT) calculate chksum with signed char,
   which will be different if there are chars in the buffer with
   the high bit set. So we calculate two checksums, unsigned and
   signed.
\end{DoxyVerb}
