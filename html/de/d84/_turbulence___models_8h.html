Detailed Description

Header file for RANS turbulence models implementation.

Author
Ramesh Kolluru
Date
2025-01-XX

This file contains the declarations for K-epsilon and K-omega turbulence models for 2D compressible flow simulations. The implementation includes:

  • Standard K-epsilon model
  • Standard K-omega model
  • Wilcox K-omega model
  • SST K-omega model
