Detailed Description

Implementation of K-omega turbulence model for 2D compressible flows.

Author
Ramesh Kolluru
Date
2025-01-XX

This file implements the K-omega two-equation turbulence model including:

  • Standard Wilcox K-omega model
  • SST (Shear Stress Transport) K-omega model

The model solves transport equations for turbulent kinetic energy (k) and specific dissipation rate (omega).

Model Equations: ∂k/∂t + ∇·(ρuk) = ∇·[(μ + σ*μt)∇k] + Pk - β*ρkω ∂ω/∂t + ∇·(ρuω) = ∇·[(μ + σ*μt)∇ω] + α(ω/k)Pk - βρω²

where μt = ρk/ω (for standard model)
