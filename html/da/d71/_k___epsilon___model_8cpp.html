Detailed Description

Implementation of K-epsilon turbulence model for 2D compressible flows.

Author
Ramesh Kolluru
Date
2025-01-XX

This file implements the standard K-epsilon two-equation turbulence model for Reynolds-Averaged Navier-Stokes (RANS) simulations. The model solves transport equations for turbulent kinetic energy (k) and turbulent dissipation rate (epsilon).

Model Equations: ∂k/∂t + ∇·(ρuk) = ∇·[(μ + μt/σk)∇k] + Pk - ρε ∂ε/∂t + ∇·(ρuε) = ∇·[(μ + μt/σε)∇ε] + C1(ε/k)Pk - C2ρ(ε²/k)

where μt = ρCμk²/ε
