__device__ void compute_flux_jacobian_device ( const double * cell_state,
double * jacobian,
int face_direction,
double face_area )

Device function to compute flux Jacobian matrix for Euler equations.

Computes the flux Jacobian ∂F/∂U for 2D Euler equations where F is the flux vector and U is the conservative variable vector [ρ, ρu, ρv, E]. The Jacobian is computed for either x-direction or y-direction faces based on the face_direction parameter.

Parameters
[in]cell_stateConservative variables [rho, rho*u, rho*v, E]
[out]jacobianOutput 4x4 Jacobian matrix (flattened row-major)
[in]face_directionFace direction: 0=left, 1=bottom, 2=right, 3=top
[in]face_areaFace area for scaling the Jacobian contribution
Note
This function assumes ideal gas with γ = 1.4 (air)
The Jacobian is scaled by face_area and includes proper sign for face direction
Includes safeguards against division by zero and negative pressure
