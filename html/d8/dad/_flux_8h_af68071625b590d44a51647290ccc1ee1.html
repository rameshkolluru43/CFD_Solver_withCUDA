void RICCA_2O ( const int & Cell_No,
int & N_Cell_No,
const int & Face_No )

Computes the dissipative flux for a given cell and face using the second-order RICCA method.

This function calculates the dissipative flux based on the left and right state variables of a cell interface. It uses the second-order RICCA method to evaluate the flux differences and applies conditions for smooth regions.

Parameters
Cell_NoThe index of the current cell.
N_Cell_NoThe index of the neighboring cell.
Face_NoThe index of the face being processed.
