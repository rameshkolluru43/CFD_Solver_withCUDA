9 * First-Order Roe Scheme Implementation
10 *
11 * This function implements the classical first-order Roe approximate Riemann solver
12 * for computing dissipative fluxes. The Roe scheme provides excellent shock-capturing
13 * capabilities by linearizing the Euler equations around Roe-averaged states.
14 *
15 * Mathematical Framework:
16 * 1. Compute Roe averages using density-weighted averaging
17 * 2. Calculate eigenvalues and eigenvectors of the linearized flux Jacobian
18 * 3. Decompose state differences into characteristic variables
19 * 4. Apply upwind dissipation based on wave speeds
20 *
21 * Features:
22 * - Exact resolution of contact discontinuities in 1D
23 * - Sharp shock capturing without spurious oscillations
24 * - Entropy fix for sonic points to prevent rarefaction shocks
25 * - Robust performance across all Mach number regimes

200 // Eigenvalues (wave speeds) - these determine the direction of information propagation
201 Lambda1 = fabs(Un - Roe_a); // Left-running acoustic wave speed
202 Lambda2 = fabs(Un); // Entropy wave speed
203 Lambda3 = fabs(Un); // Shear wave speed
204 Lambda4 = fabs(Un + Roe_a); // Right-running acoustic wave speed
205
206 // Apply entropy fix to prevent expansion shocks (rarefaction shocks)
207 // This is crucial for physical consistency near sonic points
208 double entropy_fix = 0.1 * Roe_a; // Entropy fix parameter
209
210 if (Lambda1 < entropy_fix)
211 {
212 Lambda1 = 0.5 * (Lambda1 * Lambda1 / entropy_fix + entropy_fix);
213 }
214 if (Lambda4 < entropy_fix)
215 {
216 Lambda4 = 0.5 * (Lambda4 * Lambda4 / entropy_fix + entropy_fix);
217 }
218 // Compute dissipative flux using Roe's approximate Riemann solver
219 // D = 0.5 * Σₖ |λₖ| αₖ Rₖ
220 // This represents the upwind dissipation that stabilizes the central difference
221
222 Dissipative_Flux[0] = 0.5 * (Lambda1 * alpha_1 * R11 + Lambda2 * alpha_2 * R21 + Lambda3 * alpha_3 * R31 + Lambda4 * alpha_4 * R41) * dl;

233 * Mathematical Foundation:

236 * - Applies upwind dissipation based on wave propagation direction
